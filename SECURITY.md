# Security Policy

## Supported Versions

We support the most recent released version of the project. Older versions may not receive security fixes.

## Reporting a Vulnerability

Please report security issues privately to **ricardo.montanana@example.com**.

Include as much detail as possible:
- A description of the issue and its impact
- Steps to reproduce or proof of concept
- Affected versions and environment details

We will acknowledge receipt, investigate promptly, and provide a timeline for the fix and coordinated disclosure.
